# Projektuppgift i kursen Grafisk Teknik för Webb

## Sammanfattning
I den här uppgiften skulle jag skapa en e-handelswebbplats åt ett fiktivt företag. Allt grafiskt material designade jag själv, förutom sociala ikoner som Instagram. Först designade jag webbplatsen i Figma och sedan kodade jag den med HTML, CSS och JavaScript. Webbplatsen visar bl.a. produkter, en varukorg med varor och tom samt en checkout-sida med pris och frakt.  

## Teknik som användes
- Figma
- HTML  
- CSS  
- JavaScript  
- Fetch API  
- JSON-fil för produktdata  
- LocalStorage  
- HandBreak
- Canva (skapade video)  
- Squoosh (bildkomprimering)  
- Hero Icons  
- Lighthouse

## Webbplatsen
Webbplatsen skapades åt ett fiktivt företag som heter **CanvasCrafts**. Två artister säljer sina tavlor. Eftersom fem av deras nymålade tavlor fick mycket beröm av familj och vänner valde paret att skapa en webbplats för att börja sälja sin konst och även specialbeställd konst.  

## Funktionalitet
- Tom varukorg  
- Varukorg med varor  
- Lägg till produkter i varukorgen  
- Inkrementera och dekrementera antal produkter i varukorgen  
- Radera alla produkter med ett klick  
- Visa antal produkter i varukorgen  
- Visa pris och kvantitet för varje produkt  
- Visa totalpris  
- Sammanfattning av varukorg på checkout-sidan med pris + frakt
- Skapa produktdata dynamiskt beroende på produkten

## Inspiration
Webbplatsens varukorg och struktur är inspirerade av **Audiophile challenge** från Frontend Mentors.  

## Vad jag har lärt mig
- Att testa koden tidigt är viktigt  
- Organisera koden bättre  
- Använd tydliga namn på funktioner, element och variabler  
- Undvik för stora CSS- och JS-filer  
- Det är bättre att dela upp i flera filer och minifiera dem (t.ex. med Vite)
